# vi-select2

A React library template which integrated all the things you need.

## Functionalities

- Build library code
- Run testing site
- TypeScript
- Code formatter
- Code linter
- Testing environment

## Getting start

```shell
# Clone the repo.
git clone https://github.com/musicq/vi-select2.git

# install dependencies
yarn

# install example dependencies
cd example && yarn

# back to root path
cd ..

# compile library code
yarn build

# start another window to run testing portal
yarn start
```

### Build library

```shell
yarn build

# or watching
yarn build:w
```

### Test

```shell
yarn test

# or watching
yarn test:w
```
